# docker-ce. `systemd-journal` lets the deploy user read `journalctl`
# without sudo. `adm` is the historical "can read logs" group on
# Debian/Ubuntu.
#
# Re-running is idempotent (usermod -aG; groupadd -f is a no-op when
# the group already exists).
